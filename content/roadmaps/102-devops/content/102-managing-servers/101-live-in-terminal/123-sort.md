# sort

`sort`命令用于按特定顺序对文件内容进行排序。默认情况下，它对文件的排序是假定其内容是ASCII格式的。但它也可以通过使用适当的选项来进行数字排序。

它的语法如下
`$ sort [options].. input-file` e.g. `$ sort file.txt`
