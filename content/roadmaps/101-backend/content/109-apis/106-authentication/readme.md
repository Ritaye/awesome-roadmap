# 认证

API认证过程通过使用一个认证协议来验证试图进行连接的客户的身份。该协议将请求连接的远程客户端的凭证以纯文本或加密的形式发送给远程访问服务器。然后，服务器知道它是否可以授予该远程客户端访问权。

以下是常见的认证方式的列表。

- JWT认证
- 基于令牌的认证
- 基于会话的认证
- 基本认证
- OAuth - 开放授权
- SSO--单点登录
