# OpenID

OpenID是一个利用OAuth 2.0的授权和认证机制的协议，现在已被互联网上的许多身份提供者广泛采用。
它解决了需要在许多不同的网络服务（如网上商店、讨论区等）之间分享用户的个人信息的问题。
