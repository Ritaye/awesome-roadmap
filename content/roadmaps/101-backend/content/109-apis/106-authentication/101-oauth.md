# OAuth

OAuth是**O**pen **Auth**orization 的缩写，是一种开放的授权标准。它的作用是使用访问令牌而不是用户凭证来授权设备、API、服务器和应用程序，被称为 "安全委托访问"。

在其最简单的形式中，OAuth将认证委托给Facebook、Amazon、Twitter等服务，并授权第三方应用程序访问用户账户，***无需输入其登录名和密码。

它主要用于REST/API，只提供用户数据的有限范围。
