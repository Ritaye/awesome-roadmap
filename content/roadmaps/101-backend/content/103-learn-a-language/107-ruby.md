# Ruby

Ruby是一种高级的解释性编程语言，融合了Perl、Smalltalk、Eiffel、Ada和Lisp。RUBY专注于简单性和生产力，以及自然的读写语法。Ruby支持程序性、面向对象和功能性编程，并且是动态类型的。
