# 什么是HTTP?

HTTP是基于`TCP/IP`的应用层通信协议，它将客户和服务器之间的通信方式标准化。HTTP遵循经典的 "Client-Server模型"(客户端-服务器模型)，客户端打开一个连接请求，然后等待，直到它收到一个响应。HTTP是一个无状态协议，这意味着服务器在两个请求之间不保留任何数据（状态）。
