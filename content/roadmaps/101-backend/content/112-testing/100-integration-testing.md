# 集成测试

集成测试是一个广泛的测试类别，其中多个软件模块被__集成__并作为一个整体进行测试。它的目的是测试多个服务、资源或模块之间的__互动__。例如，一个API与后端服务的交互，或一个服务与数据库的交互。
