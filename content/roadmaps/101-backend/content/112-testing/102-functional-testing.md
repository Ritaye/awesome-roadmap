# 功能测试

功能测试是指对软件进行测试，以确保满足功能要求。通常，它是一种黑盒测试，测试人员不了解源代码；测试是通过提供输入和比较预期/实际输出来进行的。
它与非功能测试形成对比，后者包括性能、负载、可扩展性和渗透测试。
