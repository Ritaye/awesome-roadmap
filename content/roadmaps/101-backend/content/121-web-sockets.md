# Web sockets

网络套接字被定义为服务器和客户之间的双向通信，这意味着双方在同一时间进行通信和交换数据。该协议从根本上定义了全双工通信。网络套接字在为网络浏览器带来丰富的桌面功能方面向前迈出了一步。
