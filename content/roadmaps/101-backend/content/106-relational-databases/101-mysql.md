# MySQL

MySQL是一个非常受欢迎的开源关系型数据库管理系统（RDBMS）。MySQL可以作为一个独立的客户端使用，也可以与其他服务结合起来提供数据库连接。LAMP堆栈中的**M**代表MySQL；仅此一点就可以看出它的普遍性。
