# 关系型数据库

关系型数据库是**一种数据库，它存储并提供对相互关联的数据点的访问**。关系型数据库将数据存储在一系列的表中。表之间的相互联系被指定为外键。外键是关系表中的一行到另一表中的唯一引用，该表可以是同一表，但最常见的是不同的表。
